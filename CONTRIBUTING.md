# Hướng dẫn đóng góp

Cảm ơn bạn muốn đóng góp cho Python Journey! Dưới đây là cách tham gia:

## Cách đóng góp

### Báo lỗi
1. Vào tab **Issues** trên GitHub
2. Bấm **New Issue**
3. Mô tả lỗi: file nào, dòng nào, lỗi gì

### Đề xuất cải tiến
1. Fork repo này
2. Tạo branch mới: `git checkout -b feature/ten-tinh-nang`
3. Commit changes: `git commit -m "Add: mô tả ngắn"`
4. Push và tạo Pull Request

### Quy tắc code
- Viết comment bằng tiếng Việt
- Tuân theo PEP 8
- Mỗi exercise file phải có docstring
- Test trước khi submit

### Cấu trúc commit message
```
Add: thêm bài tập tuần 5
Fix: sửa lỗi ex02 tuần 3
Update: cập nhật README tuần 7
Docs: thêm cheatsheet OOP
```

## Code of Conduct
- Tôn trọng mọi người
- Hỗ trợ người mới
- Không spam, không quảng cáo
