# Math Toolkit 🧮

Bộ công cụ toán học đa năng.

## Yêu cầu
1. Menu: Tính diện tích, Chu vi, Thống kê, Chuyển đổi, Thoát
2. Diện tích: hình tròn, tam giác, chữ nhật, thang
3. Thống kê: nhập dãy số → min, max, mean, median
4. Chuyển đổi: km↔miles, °C↔°F, kg↔lbs
5. Mỗi chức năng là một hàm riêng, có docstring
