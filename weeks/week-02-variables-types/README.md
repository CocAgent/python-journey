# Tuần 02: Biến & Kiểu dữ liệu 📦

> *"Đặt tên tốt là kỹ năng quan trọng nhất trong lập trình." — Robert C. Martin*

## 🎯 Mục tiêu tuần này

Sau tuần này, bạn sẽ:

- Khai báo và sử dụng biến trong Python
- Phân biệt 4 kiểu dữ liệu cơ bản: `int`, `float`, `str`, `bool`
- Dùng `input()` để nhận dữ liệu từ người dùng
- Chuyển đổi giữa các kiểu dữ liệu với `int()`, `float()`, `str()`
- Đặt tên biến theo chuẩn `snake_case`

## 📖 Kiến thức cần nắm

Biến, int, float, str, bool, input(), type(), type casting, snake_case
