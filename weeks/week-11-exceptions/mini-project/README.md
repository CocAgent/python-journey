# Robust Calculator 🔒

Máy tính chống crash hoàn toàn.

## Yêu cầu
1. Hỗ trợ: +, -, *, /, //, %, **
2. Xử lý mọi lỗi: nhập chữ, chia 0, overflow
3. Lịch sử tính toán (lưu kết quả gần nhất)
4. Hỗ trợ: 'quit' để thoát, 'history' xem lịch sử
5. KHÔNG BAO GIỜ crash, dù nhập gì
