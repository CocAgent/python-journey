## 🎯 Mục tiêu tuần này

Sau tuần này, bạn sẽ:

- Tạo và thao tác list: `append`, `insert`, `remove`, `pop`, `sort`

- Viết list comprehension — cách Pythonic xử lý list
- Hiểu tuple và khi nào dùng tuple thay list
- Kết hợp list với `if` để lọc dữ liệu
