# Password Generator 🔑

Tạo mật khẩu mạnh.

## Yêu cầu
1. Tùy chọn: độ dài, chữ hoa, chữ thường, số, ký tự đặc biệt
2. Tạo nhiều mật khẩu cùng lúc
3. Đánh giá độ mạnh (Yếu/TB/Mạnh/Rất mạnh)
4. Copy kết quả (hoặc lưu file)

## Gợi ý
- Dùng module `random` và `string`
- string.ascii_uppercase, ascii_lowercase, digits, punctuation
