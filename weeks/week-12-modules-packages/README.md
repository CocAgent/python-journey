## 🎯 Mục tiêu tuần này

Sau tuần này, bạn sẽ:

- Import và dùng module chuẩn Python (`math`, `random`, `datetime`, `os`)
- Tạo module Python của riêng mình
- Dùng `pip` cài đặt thư viện bên ngoài
- Hiểu virtual environment (`venv`)
- Hiểu cấu trúc package với `__init__.py`
