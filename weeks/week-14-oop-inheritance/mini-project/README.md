# Hệ thống ngân hàng 🏦

Hệ thống ngân hàng OOP với kế thừa.

## Yêu cầu
1. Class cha `Account`: so_tai_khoan, ten, so_du
2. Class `SavingsAccount`: lãi suất, tính lãi
3. Class `CheckingAccount`: hạn mức thấu chi
4. Gửi tiền, rút tiền, chuyển khoản giữa tài khoản
5. Lịch sử giao dịch
6. Lưu dữ liệu vào file JSON

## Gợi ý
- Dùng kế thừa + override
- super().__init__() trong class con
