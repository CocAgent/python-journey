# Word Counter 📊

Đếm tần suất từ trong văn bản.

## Yêu cầu
1. Nhập văn bản (nhiều dòng, kết thúc bằng dòng trống)
2. Đếm tần suất mỗi từ (không phân biệt hoa/thường)
3. In top-N từ phổ biến nhất
4. Vẽ biểu đồ thanh ASCII
5. Loại bỏ stop words (và, là, của, cho, với...)

## Ví dụ output
```
Top 5 từ:
python  ████████████ 12
code    ████████     8
học     ██████       6
dữ liệu████         4
bài     ███          3
```
