## 🎯 Mục tiêu tuần này

Sau tuần này, bạn sẽ:

- Tạo và thao tác dictionary (dict)

- Dict lồng nhau — dữ liệu phức tạp
- Chọn đúng cấu trúc dữ liệu cho bài toán
