# Đề cương khóa học Python Journey

> **15 tuần** · **7–10 giờ/tuần** · Từ Zero đến Dự án thực tế

---

## Giai đoạn 1 — Khởi đầu (Tuần 1–3)

### Tuần 01: Hello Python
- Cài đặt Python 3.11+ và VS Code
- `print()`, comments, phép tính cơ bản
- Python REPL, chạy file `.py`
- **Mini-project**: ASCII Art Generator

### Tuần 02: Biến & Kiểu dữ liệu
- Biến, gán giá trị, `snake_case`
- 4 kiểu cơ bản: `int`, `float`, `str`, `bool`
- `input()`, `type()`, type casting
- **Mini-project**: Thẻ sinh viên tự động

### Tuần 03: Câu lệnh điều kiện
- `if/elif/else`, thứ tự kiểm tra
- Toán tử so sánh và logic (`and`, `or`, `not`)
- Ternary expression, truthy/falsy
- **Mini-project**: Máy tính điểm GPA

---

## Giai đoạn 2 — Dữ liệu & Cấu trúc (Tuần 4–8)

### Tuần 04: Chuỗi (Strings)
- Indexing, slicing `[start:end:step]`
- String methods: `strip`, `split`, `join`, `replace`, `find`
- f-string formatting
- **Mini-project**: Text Analyzer

### Tuần 05: List & Tuple
- List: CRUD, `append/insert/remove/pop/sort`
- List comprehension
- Tuple, unpacking, immutability
- **Mini-project**: Quản lý Todo List

### Tuần 06: Vòng lặp
- `for` với `range()`, `enumerate()`, `zip()`
- `while` với điều kiện, `break/continue`
- Nested loops, pattern printing
- **Mini-project**: Pattern Printer

### Tuần 07: Hàm (Functions)
- `def`, `return`, parameters
- Default args, `*args`, `**kwargs`
- Scope (local/global), docstring
- **Mini-project**: Math Toolkit

### Tuần 08: Dictionary & Set
- Dict: key-value, `get()`, `items()`, `keys()`, `values()`
- Dict comprehension, nested dict
- Set: phép toán tập hợp
- **Mini-project**: Word Counter

---

## Tuần 09: Dự án giữa kỳ 🏆

Tổng hợp Tuần 1–8 vào project hoàn chỉnh. Chọn 1 trong 3 đề tài:
- Quiz Game, Expense Tracker, hoặc Text Adventure

---

## Giai đoạn 3 — Python thực tế (Tuần 10–14)

### Tuần 10: File I/O
- `open()`, read/write/append, context manager (`with`)
- CSV với module `csv`
- JSON với module `json`
- **Mini-project**: Contact Book

### Tuần 11: Xử lý lỗi (Exceptions)
- `try/except/else/finally`
- Các loại exception phổ biến
- Custom exception, `raise`
- **Mini-project**: Robust Calculator

### Tuần 12: Modules & Packages
- `import`, module chuẩn: `math`, `random`, `datetime`, `os`
- Tạo module riêng, `if __name__ == "__main__"`
- `pip`, virtual environment, `requirements.txt`
- **Mini-project**: Password Generator

### Tuần 13: OOP cơ bản
- Class, object, `__init__`, `self`
- Instance methods, `__str__`, `__repr__`
- Class variable vs instance variable
- **Mini-project**: Hệ thống thư viện

### Tuần 14: OOP nâng cao
- Kế thừa (inheritance), `super()`
- Override methods
- Đa hình (polymorphism), duck typing
- **Mini-project**: Hệ thống ngân hàng

Dự án hoàn chỉnh sử dụng toàn bộ kiến thức. Chọn 1 trong 3:
- Hệ thống ngân hàng mini
- Ứng dụng ghi chú (Notes App)
- Game Quiz Python
